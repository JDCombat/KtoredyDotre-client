# Którędy dotrę client

A client for jakdojade clone made for a school project, is somewhat functional and usable.


### Prerequisites

- Docker with compose
- OR Node.js ≥ 20

To fetch the backend
```bash
git submodules init
```

### Running in docker

```bash
docker-compose up --build
```

### Running

```bash
npm install

npm run dev # for dev server
```
